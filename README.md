Lingobot is a conversational AI bot that can help you learn different languages through conversation. 

Tools & Tech:
- Nextjs
- OpenAI GPT-3.5
- Tailwind
- Daisy UI
- React Hook Forms
- Next Auth
- Prisma
- Planet Scale
